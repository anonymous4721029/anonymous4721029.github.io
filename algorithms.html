Content of the package

The package contains:

  • Rhythm transcription based on merged-output HMM
  • Rhythm transcription based on note HMM
  • Rhythm transcription based on metrical HMM
  • Evaluation tool based on rhythm correction cost
  • MIDI file to transcr file converter
For details of the models and the evaluation measure, see our paper:

  • E. Nakamura, K. Yoshii and S. Sagayama, "Rhythm Transcription of Polyphonic Piano Music Based on Merged-Output HMM for Multiple Voices," IEEE/ACM TASLP, 25(4), pp. 794-806, 2017
